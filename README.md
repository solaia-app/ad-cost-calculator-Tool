# Ad Cost Calculator Full Project
Mobile first professional ad cost calculator tool with platform fees, tax breakdown, PDF download, real-time currency conversion.

## Usage
1. Upload all files to GitHub repository (private recommended)
2. Deploy to Vercel (import GitHub repo)
3. Visit live website
4. Enter amount, platform, tax, currency to calculate total
5. Download PDF receipt if needed

## Tech Stack
- Frontend: TailwindCSS + HTML + JS
- Backend: Vercel serverless API (Node.js)
- Currency API: exchangerate.host
- Mobile-first design, PDF download using jsPDF
